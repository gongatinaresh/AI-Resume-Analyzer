# 🚀 AI Resume Analyzer

An AI-powered web application that helps job seekers analyze their resumes against job descriptions, identify missing keywords, receive personalized improvement suggestions, generate AI-powered cover letters, and create LinkedIn-ready posts.

## 🚀 How to Run the Project

1. Clone this repository.
2. Open `index.html` in your browser.
3. Paste your resume and job description.
4. Click **Analyze Resume & Generate Cover Letter**.
